### 🚨 关键瓶颈：`technical_indicators.py` 的迁移状态

这个749行的脚本是深度分析管线的**核心依赖**（14项技术指标的母模块），但它仍硬编码了两个腾讯接口：

```
_fetch_tencent_quote()  →  http://qt.gtimg.cn/q={code}
_fetch_tencent_kline()  →  http://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param={code},day,,,{days},qfq
```

而tdx-api-server在同一时间点返回了正确数据：
```json
{"price":52.03, "open":55.99, "high":56.27, "low":50.97, "volume":454094}
```

这意味着当前所有深度分析报告中的技术指标依然走互联网抓取，并未使用本地自建的tdx数据源。
